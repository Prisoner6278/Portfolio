Detailed Info

Case 01 — Linguistic Bug: Intent/Action Mistranslation

The Korean line shifts the intent from “go retrieve the ring” to “put on a ring,” which can mislead the player’s next action.

Type: Linguistic Locale: ko-KR Area: Dialog / Quest Hint Severity: Minor
Source (EN)
Guess you have to get your ring yourselves.
Current (KO)
그러면 이제 반지를 껴야겠네
Expected (KO)
그러면 직접 반지를 가지러 가야겠네 or 그럼 반지는 당신이 직접 챙기러 가야겠네
Issue
  • Meaning shift: “get your ring” (obtain/retrieve) is translated as “wear/put on.”
  • Player impact: Objective text is correct, so progression impact is low. However, the dialog briefly suggests an incorrect character's emotion, reducing clarity and immersion.
Recommendation
  • Translate “get your ring” as 가지러 가다 / 챙기다 / 찾으러 가다 depending on context.
  • Avoid “끼다” unless the source explicitly indicates wear/put on.

Case 02 — Functional Bug: Alt+Tab Causes Program Exit

The application closes when the user switches focus with Alt+Tab. This is a high-severity stability issue because it blocks normal PC usage patterns.

Type: Functional Platform: Windows Area: Input Severity: Blocker
Environment
  • OS: Windows 11 Pro
  • Display Mode: Fullscreen
  • Repro Rate:Often

Steps to Reproduce
  1. Launch the game.
  2. During gameplay, press Alt+Tab to switch to another application.
  3. Observe the game behavior during/after the focus change.
Actual Result
The program exits (crash / unexpected termination).
Expected Result
The game should remain running and recover focus cleanly when returning.
Recommendation
  • Confirm reproduction by display mode: Fullscreen / Borderless / Windowed.
  • This issue may be hardware-specific. Please reproduce on an alternate device configuration (different components) to determine whether it is device-dependent.
